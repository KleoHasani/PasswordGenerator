## Getting Started

Clone:

```git
git clone https://github.com/KleoHasani/PasswordGenerator.git
```

Install dependencies:

```npm
npm install
```

Start:

```npm
npm start
```
### Required

- Node v14.8.0 or above
- NPM v6.14.7 or above
